# TRABAJO-FINAL---SISTEMAS-DE-INTEGRACIÓN-EMPRESARIAL

Este proyecto integra herramientas de SQL, Python y Power BI para analizar el comportamiento financiero de una empresa y segmentar clientes por riesgo.

## 📁 Estructura del Proyecto
* **sql/**: Scripts de creación de base de datos y modelo estrella.
* **analisis/**: Notebook de Python con limpieza de datos y Clustering (K-Means).
* **FINALBOARD.pbix**: Dashboard interactivo con los KPIs principales.

## 🛠️ Tecnologías utilizadas
* **SQL Server**: Modelado de datos.
* **Python (Pandas, Scikit-learn, Seaborn)**: Análisis estadístico y Machine Learning.
* **Power BI**: Visualización estratégica.
